YAK 抓包

物理机上下载安装YAK

双击启动

选择默认项目，左上角MTIM，界面下选择启动免配置

会打开一个浏览器，推荐chrome，其他还需要配置

输入目标地址开始劫持

输入用户名和密码，提交，成功登录后，按 F12 打开开发者工具，找到 application ，或者切换为中文，找到应用，接着找到左边目录树下的 cookie ，点击，右边会出现 cookie 详情，一个 username ，一个 password ，一个 PHPSESSID ，主页会分别显示 username ，password 的值，还有 PHPSESSID 的值中的用户名值。修改 username 的值即可实现界面上显示两个不同的用户名 admin 和dumb 。
